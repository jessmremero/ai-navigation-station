# AI导航站

## 项目概述
这是一个基于WebStack的AI导航站项目，旨在为用户提供便捷的AI相关资源导航。

## 核心功能
- 分类导航：提供多个分类的AI相关网站导航
- 响应式设计：支持各种设备访问
- 多语言支持：支持中文和英文界面

## 部署选项
1. 使用GitHub Pages部署
2. 使用Vercel部署
3. 使用Cloudflare Pages部署

## 项目结构
```
ai-navigation-station/
├── assets/
│   ├── images/
│   │   └── logos/    # 网站图标
│   ├── css/         # 样式文件
│   └── js/          # JavaScript文件
├── cn/              # 中文版
├── en/              # 英文版
└── README.md        # 项目说明文档
```

## 使用说明
1. 克隆项目到本地
2. 根据需要修改导航内容
3. 选择部署方式并部署
